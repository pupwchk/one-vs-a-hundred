# 집단지성 vs. 전문가: 주가 반응 예측 정확성 비교
> 저성능 LLM 집단과 단일 고성능 LLM의 주가 반응 예측 정확성 비교 연구

## 📋 개요

본 연구는 금융 이벤트 예측에서 **단일 전문가 모델(GPT-5)**과 **페르소나 기반 집단지성 모델(GPT-5-nano)**의 성능을 체계적으로 비교·분석합니다. 복잡한 금융 시장 예측에서 다양한 관점이 결합된 집단지성이 단일 전문가 모델보다 더 높은 예측력을 보일 수 있음을 실증적으로 입증합니다.

### 주요 연구 결과

- **집단지성 모델 정답률**: 71.29%
- **전문가 모델 정답률**: 64.19%
- **통계적 유의성**: p < 0.01
- **평균 수익률**: +7.37% vs. +4.81%
- **가중점수**: +40.21 vs. +19.34

## 🎯 연구 질문

> 복잡한 금융 시장 예측에서 단일 전문가 모델의 깊이 있는 논리와 다수의 다양한 관점이 결합된 집단지성 모델 중 어느 것이 더 높은 예측력을 보이는가?

## 🏗️ 연구 방법론

### 두 가지 모델 접근법

#### 1. 전문가 모델 (단일 고성능 LLM)
- **모델**: GPT-5
- **기법**: Chain-of-Thought (CoT) 프롬프팅
- **특징**: 단계적 논리 추론을 통한 심층적 분석 수행

#### 2. 집단지성 모델 (다수의 저성능 LLM)
- **모델**: 25개의 독립적인 GPT-5-nano 에이전트
- **페르소나**: European Social Survey (ESS) 데이터 기반
- **의사결정 방식**: 확신도 가중 평균
- **핵심 특징**: 실제 투자자들의 이질적 특성을 반영한 다양한 사고 구조

### 페르소나 차원

각 sLLM 에이전트는 4가지 차원으로 구성된 고유한 페르소나를 부여받습니다:

1. **Identity (정체성)**: 국적, 연령, 성별, 교육 수준
2. **Socio-Economics (사회경제)**: 직업, 소득 분위, 정치적 성향
3. **Outlook and Trust (전망과 신뢰)**: 행복도, 국가 경제 전망, 대인/제도 신뢰도
4. **Information Habit (정보 습관)**: 정치 관심도, 뉴스 및 인터넷 사용 빈도

## 📊 데이터셋

- **대상**: S&P 100 기업
- **이벤트 수**: 124건 (주가 등락폭 10% 이상)
- **예측 시점**: 1일, 3일, 7일, 15일, 30일 후
- **총 예측 횟수**: 620회 (모델당)

### 데이터 수집

- **주가 데이터**: Yahoo Finance API
- **뉴스 데이터**: Google RSS 기반 GNews 라이브러리
- **페르소나 데이터**: European Social Survey Round 11 (2022)

## 🔬 실험 설계

### 전문가 모델
```
시스템 프롬프트: CoT 기반 단계적 추론
1. 이벤트 해석
2. 유사 사례와의 비교
3. 잠재적 요인 도출
4. 최종 결론 도출
```

### 집단지성 모델
```
25개 독립 에이전트 → 각자 예측 및 확신도 제시 → 확신도 가중 평균 → 최종 결정
```

## 💡 주요 발견

1. **단기 예측 우위**: 집단지성 모델이 특히 1-7일 단기 예측에서 두드러진 성능 우위
2. **장기 예측 수렴**: 30일 장기 예측에서는 두 모델의 정답률이 수렴하는 경향
3. **확신도와 수익률 상관**: 높은 확신도가 실제 수익률 증가로 이어짐 (r = 0.17~0.23)
4. **통계적 유의성**: 모든 주요 지표에서 집단지성 모델의 우위가 통계적으로 유의미

## 🔍 연구 의의

### 이론적 기여
- LLM을 활용한 금융 의사결정에서 **모델 규모 확대보다 이질적 사고 구조의 통합**이 더 큰 예측 효율성을 가져올 수 있음을 실증
- 집단지성 이론을 AI 시스템에 성공적으로 적용한 사례 제시

### 실무적 시사점
- **AI 오케스트레이션**의 중요성: 더 큰 단일 모델보다 다수의 소형 에이전트를 효과적으로 구성하는 것이 더 효율적일 수 있음
- 행동경제학적 다양성을 AI 시스템에 반영하여 시장의 복잡성과 비합리성을 더 잘 포착 가능


## 🔮 향후 연구 방향

1. **모델 간 상호작용**: 현재는 독립적 구조지만, 토론과 정보 교환을 통한 협의 과정 구현
2. **계층적 집단 프레임워크**: 다층 구조의 집단지성 시스템 설계
3. **하이브리드 접근**: 전문가 모델의 논리적 일관성과 집단지성 모델의 다양성을 결합한 프레임워크
